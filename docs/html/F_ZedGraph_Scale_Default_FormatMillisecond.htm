ScaleDefaultFormatMillisecond Field
A default setting for the Date auto-ranging code. This values applies only to Date-Time type axes. This is the format used for the scale values when auto-ranging code selects a Format of Millisecond for MajorUnit and Millisecond for for MinorUnit. This value is used by the CalcDateStepSize(Double, Double, Scale) method.
