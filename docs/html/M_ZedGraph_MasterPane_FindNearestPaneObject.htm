Remarks
This method first finds the GraphPane within the list that contains the specified mouse point. It then calls the FindNearestObject(PointF, Graphics, Object, Int32) method to determine which object, if any, was clicked. With the exception of the , all the parameters in this method are identical to those in the FindNearestObject(PointF, Graphics, Object, Int32) method. If the mouse point lies within the Rect of any GraphPane item, then that pane will be returned (otherwise it will be null). Further, within the selected pane, if the mouse point is within the bounding box of any of the items (or in the case of ArrowObj and CurveItem, within NearestTol pixels), then the object will be returned. You must check the type of the object to determine what object was selected (for example, "if ( object is Legend ) ..."). The parameter returns the index number of the item within the selected object (such as the point number within a CurveItem object.
