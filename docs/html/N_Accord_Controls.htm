Accord.Controls Namespace
Contains useful Windows Forms controls to help creating scientific applications. Includes histograms, scatterplots, wavecharts and tabular data viewers for windows forms applications.

Remarks

Useful controls offered in this namespace are the DataGridBox, DataSeriesBox and ScatterplotBox controls. Those controls can be used to create MessageBox-like user dialogs presenting data to the user. If the Controls.Audio assembly is also loaded, it is also possible to use the same controls to show wavecharts for audio signals.

Other very useful feature is the ArrayDataView class, which enables any array, multi-dimensional matrices or jagged-matrices to be data-bounded inside a DataGridView and other similar controls.
