MasterPaneSetLayout Method
The SetLayout() methods setup the desired layout of the GraphPane objects within a MasterPane. These functions do not make any changes, they merely set the parameters so that future calls to ReSize(Graphics, RectangleF) or DoLayout(Graphics) will use the desired layout.

The layout options include a set of "canned" layouts provided by the PaneLayout enumeration, options to just set a specific number of rows and columns of panes (and all pane sizes are the same), and more customized options of specifying the number or rows in each column or the number of columns in each row, along with proportional values that determine the size of each individual column or row.
