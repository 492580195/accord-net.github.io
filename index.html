Scientific computing made easy

The Accord.NET Framework is a complete framework for building machine learning, computer vision, computer audition, signal processing and statistical applications. Sample applications provide a fast start to get up and running quickly, and an extensive documentation helps fill in the details.

Sample applications.

Sample applications help you start writing your applications. Just get one of the sample applications that is closest to your goal, and start from there.

Create and test new learning algorithms with ease.

Strategy and template method patterns help you swap learning algorithms quickly. Create, build and compare different approaches without delving too deep in code.

Discuss and share.

With documentation, discussion groups and an ever willing wish to accept new code contributions, make your work visible by sharing and discussing with the community.
